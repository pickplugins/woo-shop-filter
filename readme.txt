	Filter shop product by advance filter and search.

== Description ==

Default WooCommerce shop page isn't good enough for search and filter product as user needs, this plugin allows user to filter and search by their own needs and find the best product they are looking for.

### WooCommerce Shop Filter by [http://pickplugins.com](http://pickplugins.com)


### Plugin Features

* Filter by keywords
* Filter by product categories
* Filter by product tags
* Filter by price range
* Filter by order & order by
* Filter onsale product
* Filter in-stock
* Filter by SKU

### How to use?

Please use folloing shortcode via text widget on sidebar

`[WCShopFilter]`









== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>WooCommerce Shop Filter</strong>" activate it.<br />
